# LILA Player Journey Visualization Tool

This tool visualizes player movement and combat events on game maps to help Level Designers understand player behavior.

## Features

- Player path visualization
- Human vs bot detection
- Event markers (kills, deaths, loot, storm)
- Timeline playback
- Heatmap overlays
- Match filtering

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
